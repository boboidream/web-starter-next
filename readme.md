# Web-starter-next

## Use
```bash
yarn install
yarn run dev
```

## Features
gulp@4.0

gulpfile with es6

rollup tree shaking

swig -> *.html
less -> *.css
es2015,es2016,es2017 -> es5.js

---
## 特性
将 `src` 下的 `.less` 文件，转为 `.css` 文件并添加浏览器前缀，忽略 `_*.less` 类型模板文件。

webpack-stream + webpack2

js -> bundle.js
